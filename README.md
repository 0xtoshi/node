# Tutorial

1. Install Nodejs
2. Install Adonis CLI
``` npm i -g @adonisjs/cli ```
3. Start Mysql
4. Edit kro rename .env.example dadi .env tok
5. Edit bagian DB_CONNECTION=sqlite dadi DB_CONNECTION=mysql
6. Migrasi
`` adonis key:generate ``
`` adonis migration:run ``

7. Letsgo Start 

``` adonis serve -- dev ```

Buka 

http://127.0.0.1:3333/dashboard
